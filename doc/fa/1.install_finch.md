# نصب

Finch یک فریم‌ورک وب سمت سرور برای دارت است. این صفحه پیش‌نیازها و مراحل نصب آن را پوشش می‌دهد.

## پیش‌نیازها

- **دارت SDK** — فینچ نیازمند دارت ۳.x یا بالاتر است. از [dart.dev/get-dart](https://dart.dev/get-dart) دانلود کنید.
- یک ترمینال و ویرایشگر متن (VS Code با افزونه Dart توصیه می‌شود).

## نصب Finch CLI

Finch CLI ابزارهای قالب‌بندی پروژه، ساخت، سرویس، و مدیریت مهاجرت دیتابیس را فراهم می‌کند.

```bash
dart pub global activate finch
```

پس از فعال‌سازی، صحت نصب را تأیید کنید:

```bash
finch --version
```

## ساخت پروژه جدید

```bash
finch create
```

از شما خواسته می‌شود نام پروژه، مسیر، و قالب را انتخاب کنید. قالب `example` شامل نمونه‌های MongoDB، MySQL، SQLite، WebSocket، i18n، فرم‌ها، و مستندات API است.

## افزودن Finch به پروژه موجود

```bash
dart pub add finch
```

سپس اجرا کنید:

```bash
dart pub get
```

## مرحله بعد

برای تمام دستورات موجود به [Finch CLI](2.finch_cli.md) مراجعه کنید، یا مستقیماً به [پیکربندی](3.configuration.md) بروید تا اولین برنامه خود را راه‌اندازی کنید.

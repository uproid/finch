# Nginx voor Finch

Finch is eenvoudig te gebruiken met Docker en Nginx. Je kunt het `nginx.conf` bestand in de `example` map gebruiken om je Finch-applicatie met Nginx te draaien.

In de `example` map vind je een `nginx.conf` bestand dat je kunt gebruiken voor het uitvoeren van het voorbeeldproject met Nginx.

# Finch Installatiegids

Welkom bij de Finch Installatiegids! Deze gids begeleidt je bij het installeren en instellen van Finch, een krachtig webframework voor Dart. Of je nu een ervaren ontwikkelaar bent of net begint, Finch biedt krachtige hulpmiddelen om server-side webapplicatieontwikkeling te vereenvoudigen.

## Vereisten
Voordat je begint, zorg ervoor dat je de volgende vereisten op je systeem hebt geïnstalleerd:

1. **Dart SDK**: Finch is gebouwd met Dart, dus je moet de Dart SDK geïnstalleerd hebben. Je kunt deze downloaden vanaf de [officiële Dart-website](https://dart.dev/get-dart).
2. **Teksteditor**: Kies je favoriete teksteditor of IDE. Populaire opties zijn Visual Studio Code, IntelliJ IDEA of een andere editor die Dart ondersteunt.
3. **Terminal**: Je hebt een terminal of opdrachtprompt nodig om Dart-opdrachten uit te voeren.

## Installatiestappen
Installeer Finch met de volgende opdracht:
```bash
dart pub global activate finch
```

## Maak een nieuw project
Maak een nieuw project aan met de volgende opdracht:
```bash
finch create
```

## Finch toevoegen aan een bestaand project
Voeg Finch toe aan je bestaande project met de volgende opdracht:
```bash
dart pub add finch
```

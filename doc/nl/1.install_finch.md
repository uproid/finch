# Installatie

Finch is een server-side webframework voor Dart. Deze pagina beschrijft de vereisten en de installatiestappen.

## Vereisten

- **Dart SDK** — Finch vereist Dart 3.x of hoger. Download via [dart.dev/get-dart](https://dart.dev/get-dart).
- Een terminal en een teksteditor (VS Code met de Dart-extensie wordt aanbevolen).

## Finch CLI installeren

De Finch CLI biedt project-scaffolding, build-, serve- en databasemigratieopdrachten.

```bash
dart pub global activate finch
```

Controleer na de activering de installatie:

```bash
finch --version
```

## Een nieuw project aanmaken

```bash
finch create
```

Je wordt gevraagd een projectnaam, pad en sjabloon te kiezen. Het `example`-sjabloon bevat voorbeelden van MongoDB, MySQL, SQLite, WebSocket, i18n, formulieren en API-documentatie.

## Finch toevoegen aan een bestaand project

```bash
dart pub add finch
```

Voer vervolgens uit:

```bash
dart pub get
```

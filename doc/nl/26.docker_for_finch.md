# Docker voor Finch

Finch is geschikt voor gebruik met Docker en Nginx. Je kunt het project uitvoeren met een `docker-compose.yaml` bestand en een `Dockerfile`.

In de `example` map vind je een `docker-compose.yaml` en een `Dockerfile`. Gebruik deze bestanden om het voorbeeldproject met Docker en Nginx te draaien.

[docker-compose.yaml](https://github.com/uproid/finch/blob/master/example/docker-compose.yaml)

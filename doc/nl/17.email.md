# E-mail verzenden Gids

Welkom bij de Finch E-mail verzenden Gids! Deze gids laat zien hoe je e-mails verstuurt vanuit je Finch-applicatie. Of je nu een ervaren ontwikkelaar bent of net begint, Finch maakt e-mailfunctionaliteit eenvoudig.

## E-mail verzenden
De `MailSender` klasse wordt gebruikt om e-mails te versturen in je Finch-applicatie. Hieronder staat een voorbeeld van het gebruik:
```dart
import 'package:finch/src/mail/mail_sender.dart';

bool res = await MailSender.sendEmail(
  from: 'sender@example.com',
  fromName: 'Sender Name',
  host: 'smtp.example.com',
  port: 587,
  to: ['recipient@example.com'],
  subject: 'Test Email',
  html: '<h1>Hello, World!</h1>',
  text: 'Hello, World!',
  username: 'smtp-username',
  password: 'smtp-password',
  ssl: true,
  allowInsecure: false,
);
```

# Debugging Gids

Welkom bij de Finch Debugging Gids! Deze handleiding legt uit hoe je debugging gebruikt in je Finch-applicatie. Finch biedt hulpmiddelen om je applicatie in realtime te debuggen.

Debugging in Finch is een krachtig hulpmiddel waarmee je je applicatie in real-time kunt debuggen. Finch biedt een consolewidget en (aan de frontend) een debugbalk onderaan de pagina. Met deze balk bekijk je logs, fouten en andere informatie. Je kunt de balk ook gebruiken om de pagina te herladen, de server te herstarten, routes te controleren en andere nuttige acties uit te voeren.

### Opmerking:
Debugging is alleen beschikbaar in de lokale omgeving. Het is niet beschikbaar in productie. Zet `config.isLocalDebug` op `true` in je `app.dart` en zet ook `config.enableLocalDebugger` op `true` om debugging functionaliteit te gebruiken.

## 功能特性

1. 使用 OpenAPI 文档快速开发 API
2. 支持实时应用的 WebSocket
3. MongoDB、MySQL 和 SQLite 数据库集成
4. 计划任务（定时任务）
5. 多语言支持
6. 开发调试工具
7. 数据库架构管理的迁移系统
8. 具有验证和错误处理的高级表单
9. SMTP 邮件支持
10. CSS、JS 和图像的资源管理
11. HTML 渲染的模板引擎
12. 用于实时应用的 WebSocket 支持
13. 用于生成 HTML 内容的 HTMLer
14. 用于生成 SQL 查询的 Sqler
15. API 的 OpenAPI 文档
16. 更多功能...

# 安装

Finch 是一个基于 Dart 的服务端 Web 框架。本页介绍安装的前提条件和步骤。

## 前提条件

- **Dart SDK** — Finch 需要 Dart 3.x 或更高版本。从 [dart.dev/get-dart](https://dart.dev/get-dart) 下载。
- 一个终端和文本编辑器（推荐使用带有 Dart 扩展的 VS Code）。

## 安装 Finch CLI

Finch CLI 提供项目脚手架、构建、服务和数据库迁移命令。

```bash
dart pub global activate finch
```

激活后，验证安装是否成功：

```bash
finch --version
```

## 创建新项目

```bash
finch create
```

系统会提示您选择项目名称、路径和模板。`example` 模板包含 MongoDB、MySQL、SQLite、WebSocket、i18n、表单和 API 文档示例。

## 将 Finch 添加到现有项目

```bash
dart pub add finch
```

然后运行：

```bash
dart pub get
```
